# Logo LPLAN

## Como adicionar a logo

1. Coloque o arquivo da logo da LPLAN neste diretório com o nome:
   - `lplan-logo.png` (formato PNG recomendado)
   - OU `lplan-logo.svg` (formato SVG)

2. A logo será exibida automaticamente na página de boas-vindas (home) quando o usuário não estiver logado.

3. Tamanho recomendado:
   - Largura máxima: 120px
   - Altura: proporcional (será ajustada automaticamente)
   - Formato: PNG com fundo transparente ou SVG

## Localização do arquivo

O sistema procurará primeiro por `lplan-logo.png` e, se não encontrar, tentará `lplan-logo.svg`.

## Onde a logo aparece

- Página inicial (home) quando o usuário não está autenticado
- Substitui o ícone anterior (documento/planilha)
